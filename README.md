# Customer Demographics Analysis

## Project Overview

This project analyzes customer demographic data using Python.

### Analysis Performed
- Total Customers Count
- Average Age Calculation
- Gender Distribution Analysis
- City-wise Customer Count
- Highest Age Identification
- Lowest Age Identification
- Data Visualization using Pie Charts and Bar Charts

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Outputs
- Gender Pie Chart
- Age Distribution Bar Chart
- City Count Analysis
- Customer Statistics Summary

## Conclusion

Customer demographic data was analyzed based on age, gender, and city. The project provides insights into customer distribution patterns through statistical analysis and visualizations.
